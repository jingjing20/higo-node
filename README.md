higo 服务端

## 生成密钥与公钥

```bash
cd config

# 使用 openssl
openssl

# 生成私钥 private.key
genrsa -out private.key 4096

# 基于私钥 private.key 生成公钥 public.key
rsa -in private.key -pubout -out public.key

# 退出
exit
```
